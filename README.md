# Whisper API
There are handler functions to provide endpoint for whisper model and python script file for testing the performance of whisper model.

## Prerequisites

Ensure you have Python installed on your machine before proceeding.

## Getting Started

Follow these steps to get the application up and running:

#### 1. Install Dependencies:

Begin by installing the necessary libraries.

```bash
python -m venv env

source env/bin/activate

pip install -r requirements.txt
```

#### 2. Launch the Server:

Once installed necessary libraries, you can run the server

```bash
python handler.py
```

After starting the server, access the application by navigating to http://0.0.0.0:7000(http://ip:7000) in your web browser.
